# f404-rtos
ULB INFO F404 Project 1 Repo

# How to use
## Windows
run `plot_all.bat` to plot all the graphs

run `python3 plot.py dm|edf|rr <taskset_path>` to plot a specific graph

## Linux

run `./plot_all.sh` to plot all the graphs

run `python3 plot.py dm|edf|rr <taskset_path>` to plot a specific graph
